Report Perjalanan 90 Hari Berhenti Merokok

Berikut adalah laporan harian mengenai kondisi saya setelah berhenti merokok selama 90 hari:

Laporan Harian

Hari ke-1 tanggal 10 Desember 2024:

Pada hari pertama, saya hampir merokok setelah bangun tidur, tetapi mengingat proyek 90 hari ini dan dukungan orang-orang terdekat, saya berhasil tetap kuat meski merasa cemas dan gelisah.

Hari ke-2 tanggal 11 Desember 2024:

Pada hari kedua, saya tidak terlalu menjalankannya dengan berat, karena saya mempunyai banyak tugas kuliah yang harus selesai pada hari itu.

Hari ke-3 tanggal 12 Desember 2024:

Pada hari ketiga, saya tidak berfikir untuk merokok, karena saya melakaukan kegiatan membaca alkahfi di kost saya, biasanya saya merokok ketika membaca alkahfi dikost orang lain.

Hari ke-4 tanggal 13 Desember 2024:

Pada hari keempat, kegiatan saya masih banyak, saya tidak sempat memikirkan untuk merokok walaupun kelas kosong tapi diganti dengan seminar sampai sore.

Hari ke-5 tanggal 14 Desember 2024:

Pada hari kelima, saya berfikir untuk merokok ketika dijalan untuk pulan kedepok, tapi saya berusaha untuk menahannya agar tidak merokok.

Hari ke-6 tanggal 15 Desember 2024:

Pada hari keenam, saya merokok untuk pertama kalinya dalam seminggu, saya sudah tidak kuat lagi menahan godaan tersebut, ditambah saya mewajarkan untuk merokok, tapi saya menyesal telah kembali merokok

Hari ke-7 tanggal 16 Desember 2024:

Pada hari ketujuh, saya tidak berpikir untuk merokok karena saya sibuk dengan keluarga saya, saya menghabiskan waktu dengan adik saya.

Hari ke-8 tanggal 17 Desember 2024:

Pada hari kedelapan, saya sangat sibuk pada hari itu untuk persiapan pulang ke kost saya.

Hari ke-9 tanggal 18 Desember 2024:

Pada hari kesembilan, saya ada berfikir untuk merokok, karena banyaknya acara dan kelas yang padat saya melupakannya

Hari ke-10 tanggal 19 Desember 2024:

Pada hari kesepuluh, saya berkeinginan untuk merokok, tapi saya masih bisa menahannya.

Hari ke-11 tanggal 20 Desember 2024:

Pada hari keenam, saya merokok untuk pertama kalinya dalam seminggu, saya sudah tidak kuat lagi menahan godaan tersebut, ditambah saya mewajarkan untuk merokok, tapi saya menyesal telah kembali merokok

Hari ke-12 tanggal 21 Desember 2024:

Pada hari kedua belas, saya mendapatkan kabar bahwa ibu saya masuk rumah sakit, jadi saya butuh untuk menghasilkan sesuatu yang merangsang dopamine, dan saya memutuskan untuk merokok.

Hari ke-13 tanggal 22 Desember 2024:

Pada hari ketiga belas, saya menemani ibu saya di rumah sakit dan tidak ada kesempatan untuk merokok.

Hari ke-14 tanggal 23 Desember 2024:

Pada hari keempat belas, saya masih menemani ibu saya di rumah sakit.

Hari ke-15 tanggal 24 Desember 2024:

Pada hari kelima belas, saya pergi ke rumah, tapi karena rasa kesedihan saya yang sangat besar, saya tidak berfikir untuk merokok.

Hari ke-16 tanggal 25 Desember 2024:

Pada hari keenam belas, saya sibuk untuk mempersiapkan diri untuk berangkat ke kost.
